Cloudinary Implementation Plan

Namaste! Here is your extremely granular, step-by-step guide.

Zero Image Drops Guarantee: We are strictly changing the inside of the uploadScreenshot function. Because the rest of your app (Findings, Task Staging, Basecamp Push) treats the URL as a simple string, they won't even know it changed. The flow remains 100% unbreakable.
Automatic Deletion: We will add a small hook in the API when a "Run" is deleted to find all Cloudinary URLs for that run and wipe them instantly.

Step 1: Install Package & Configure Environment

First, we need to add the Cloudinary package to both your worker and api folders.

Run this command in your terminal: 
cd apps/worker && pnpm add cloudinary && cd ../api && pnpm add cloudinary

Add these 3 lines to your .env file anywhere: 
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

Step 2: Update Worker Storage Logic

We will replace the Supabase storage logic with Cloudinary logic. All other files that use this function will automatically start uploading to Cloudinary without any changes needed.

Open file: apps/worker/src/lib/supabaseStorage.ts

Replace from this code line: const DEFAULT_BUCKET = 'screenshots'; at line number 12
To these lines (stop before end of file): } at line number 88

Replace with this exact new code:

import { v2 as cloudinary } from 'cloudinary';

cloudinary.config({
  cloud_name: process.env.CLOUDINARY_CLOUD_NAME,
  api_key: process.env.CLOUDINARY_API_KEY,
  api_secret: process.env.CLOUDINARY_API_SECRET
});

export async function uploadScreenshot(
  buffer: Buffer, 
  path: string, 
  options: { bucket?: string; isPublic?: boolean } = {}
): Promise<string> {
  logger.info({ path }, 'Uploading screenshot to Cloudinary');
  
  const publicId = path.replace(/\.[^/.]+$/, "");

  return new Promise((resolve, reject) => {
    const uploadStream = cloudinary.uploader.upload_stream(
      { 
        public_id: publicId,
        resource_type: 'image'
      },
      (error, result) => {
        if (error) {
          logger.error({ error, path }, 'Failed to upload to Cloudinary');
          return reject(error);
        }
        if (result) {
          resolve(result.secure_url);
        }
      }
    );
    uploadStream.end(buffer);
  });
}

Step 4: Delete Images when Run is Deleted

To ensure we keep Cloudinary clean and completely free, we will delete the images right before the run gets deleted.

Open file: apps/api/src/routes/runs.ts

Replace this single line of code: const { error } = await supabase.from("qa_runs").delete().in("id", runIds) at line number 809

Replace that single line with this exact new code:

      // 1. Get findings and pages to find Cloudinary URLs
      const { data: findings } = await supabase.from("findings").select("screenshot_url").in("run_id", runIds);
      const { data: pages } = await supabase.from("pages").select("screenshot_url_desktop, screenshot_url_tablet, screenshot_url_mobile").in("run_id", runIds);

      const publicIdsToDelete: string[] = [];
      
      const extractCloudinaryId = (url: string) => {
        if (!url || !url.includes("cloudinary.com")) return;
        const match = url.match(/\/upload\/(?:v\d+\/)?(.+?)\.[a-z]+$/i);
        if (match && match[1]) publicIdsToDelete.push(match[1]);
      };

      if (findings) {
        findings.forEach((f: any) => {
          if (f.screenshot_url) f.screenshot_url.split(",").forEach(extractCloudinaryId);
        });
      }
      if (pages) {
        pages.forEach((p: any) => {
          if (p.screenshot_url_desktop) extractCloudinaryId(p.screenshot_url_desktop);
          if (p.screenshot_url_tablet) extractCloudinaryId(p.screenshot_url_tablet);
          if (p.screenshot_url_mobile) extractCloudinaryId(p.screenshot_url_mobile);
        });
      }

      // 2. Delete from Cloudinary
      if (publicIdsToDelete.length > 0) {
        const cloudinary = require("cloudinary").v2;
        cloudinary.config({
          cloud_name: process.env.CLOUDINARY_CLOUD_NAME,
          api_key: process.env.CLOUDINARY_API_KEY,
          api_secret: process.env.CLOUDINARY_API_SECRET,
        });
        
        // Cloudinary bulk delete allows max 100 per request
        for (let i = 0; i < publicIdsToDelete.length; i += 100) {
          await cloudinary.api.delete_resources(publicIdsToDelete.slice(i, i + 100));
        }
      }

      const { error } = await supabase.from("qa_runs").delete().in("id", runIds);
